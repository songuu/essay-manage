### vue新版的安装和选择

1. vue install -g @vue/cli 进行vue的更新
2. vue create project-name 新建项目
3. 出现两个选项  一：default 二：manual手动设置
4. 选项设置
> ( ) TypeScript  使用typeScript
> ( ) Progressive Web App (PWA) Support  渐进式的网页应用程序 
> ( ) Router  使用router代理路由
> ( ) Vuex   使用vuex代理系统状态
> ( ) CSS Pre-processors  css预编译
> ( ) Linter / Formatter  
> ( ) Unit Testing
> ( ) E2E Testin 
如果是大项目的话，就需要配置格式化
